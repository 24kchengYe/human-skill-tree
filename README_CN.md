> **AI 是人类有史以来最好的老师**——无限耐心、随时在线、精通一切学科。但缺乏结构化的技能引导，它只是一个什么都知道却不会教你的天才。
>
> **人类技能树** 将任何 AI 智能体变成一个结构化的、基于认知科学的学习伙伴——从幼儿园算术到博士研究，从编程面试到中国饭局礼仪，全覆盖。

---

## 为什么做这个项目

大学花 4 年传递的知识，AI 几周就能交付。课本是静态的，AI 能实时适配你的水平。但原始的 AI 对话缺乏让知识「留下来」的**教学结构**。

这个项目就是那座桥：

| 传统学习 | 裸聊 AI | 🌳 人类技能树 |
|---|---|---|
| 千人一面 | 无结构 | 自适应 + 结构化 |
| 被动（读/听） | 只有问答 | 主动回忆、间隔重复、苏格拉底对话 |
| 单一学科 | 啥都懂，不会教 | 按领域策划学习路径 |
| 止于课堂 | 无课程体系 | K-12 → 本科 → 职业 → 人生 |
| 不教做人 | 泛泛建议 | 文化敏感的情景式社交训练 |

## 基于学习科学

每个技能都遵循认知科学原理——不仅教「什么」，更关注「人类如何学习和记忆」：

| 原理 | 机制 | 技能中的应用 |
|---|---|---|
| 🔄 **间隔重复** | 对抗遗忘曲线 | 技能在最优间隔安排复习 |
| 🧪 **主动回忆** | 提取练习比重读强 10 倍 | 技能考你，而不是给你念 |
| 🎯 **合意困难** | 短期困难 = 长期收益 | 技能推你到舒适区之外 |
| 🔀 **交错练习** | 混合主题提升辨别力 | 技能在相关概念间交替 |
| 🖼️ **双重编码** | 语言 + 图像 = 更强编码 | 技能结合文字、图示、案例 |
| 🪞 **苏格拉底法** | 提问 > 讲述 | 技能通过问题引导，而非灌输 |
| 🧩 **组块化** | 信息打包为有意义单元 | 技能将复杂主题拆成可消化片段 |
| 📊 **Bloom 分类** | 6 层：记忆 → 创造 | 技能按认知复杂度递进 |

### 快速开始

```bash

git clone https://github.com/YOUR_USERNAME/human-skill-tree.git

# 复制所有技能
```

## 理念

### 🎮 为什么叫「技能树」？

在 RPG 游戏中，技能树是一个分支结构——投入点数解锁能力。现实人生也一样：

- **前置技能很重要**：先学代数才能学微积分，先有共情才能谈判
- **多条路径**：不是每个人都需要相同的技能
- **专精是有价值的**：一个方向的深度 > 所有方向的浅尝
- **可以随时「洗点」**：转行、新兴趣、终身学习

### 🤝 为什么有「人情世故」？

目前没有任何 AI 技能项目覆盖**人情世故**。但这可能是人生最重要的技能：

- 怎么察言观色
- 什么时候该说、什么时候该听
- 送礼有什么讲究
- 怎么在办公室政治中生存
- 怎么维系关系又不被消耗

我们相信 AI 可以成为一个**安全的练习场**——通过情景模拟学习这些微妙的社交技能。
