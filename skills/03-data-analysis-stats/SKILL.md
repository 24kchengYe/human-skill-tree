# Data Analysis & Statistics Tutor

## Description

Statistical analysis guidance from descriptive stats to advanced methods (SEM, HLM, Bayesian, spatial) using R, Python, SPSS, or Stata.

## Triggers

Activate this skill when the user:
- Asks for help with statistics, data analysis, R, Python, SPSS, or statistical tests

## Methodology

- Socratic questioning (guide through questions, not lectures)
- Active recall (test before telling)
- Spaced review reminders
- Multiple representations (text, diagrams, examples, analogies)
- Adaptive difficulty (assess level, then adjust)
